　１．はじめに

　　　この度、xna-morijobi-winをダウンロードしていただきありがとうございます。
　　　このxna-morijobi-winは、usagiさんのxna-morijobi-winを引用して編集、追加したものです。
　　　これについては、usagiさんに許可を得ています。

　２．インストール方法

　　　インストールは不要です。
　　　xna-morijobi-winのフォルダをダウンロードすればいつでも動作します。

　３．アンインストール方法

　　　xna-morijobi-winフォルダをフォルダごと削除してください。

　４．動作環境

　　　・Microsoft Visual C# 2010 シリーズ
　　　・XNA Game Studio 4.0

　　　どちらもMicrosoft様のサイトにて、ダウンロード・インストールできます。
